For windows,

Step 1 : Use "pip install virtualenv"
Step 2 : "run virtualenv nameofvenv" in terminal
Step 3 : activate that virtualenv (different for different os)
Step 4 : run "pip -r requirements.txt" in terminal (virtualenv should be activated) 
Step 5 : To compile libaray run "python setup.py sdist bdist_wheel" in terminal 

For other os,
Step 1 : Use pip install virtualenv
Step 2 : run virtualenv nameofvenv in terminal
Step 3 : activate that virtualenv
Step 4 : Run following command in terminal,
* python -m pip install -upgrade pip
* pip install wheel
* pip install twine
* pip install mypy
Step 5 : To compile libaray run "python setup.py sdist bdist_wheel" in terminal
